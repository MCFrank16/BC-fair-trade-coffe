# BC-fair-trade-coffe
This proves the authenticity of the coffee by using ethereum blockchain.


## to run the application follow these steps, make sure you have a metamask wallet
`npm install`
`npm run dev`

## to run the application frontend follow these steps
`cd app/src`
`npm install`
`npm run dev`

## Built with
Truffle v5.1.13

node v12.16.1

web3 v1.2.1
